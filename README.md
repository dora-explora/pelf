# Portable Executable Linkable Format

a.k.a. PELF, a binary that can run natively on both Windows and Linux!
This project is a format and corresponding utility that combines a Windows **P**ortable **E**xecutable file (commonly know by its extension .exe) and a Linux **E**xecutable **L**inkable **F**ormat file through a shebang-less shell script to make a binary [polyglot](https://wikipedia.org/wiki/Polyglot_\(computing\)) that can run on both Windows and Linux!
